# Avengers Multiplayer Quiz App

## 🚀 Overview
A real-time multiplayer quiz system built using Node.js, Express, and Socket.io.

## 🎮 Features
- Live multiplayer participation
- Admin control panel
- Phase-based scoring system
- Real-time leaderboard
- Dynamic question loading via JSON

## 🛠 Tech Stack
- Node.js
- Express
- Socket.io
- HTML, CSS, JavaScript

## ▶️ How to Run
1. npm install
2. npm start
3. Open http://localhost:3000
